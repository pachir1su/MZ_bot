# MZ_bot

Discord 도박 봇

## 주요 기능
- 10분마다 기본 코인 지급

- 베팅과 순위 확인 등 경제 시스템
- 관리자 전용 설정 및 질의응답 기능

- 도박이 계속 성공하나, 잔액이 변경되지 않는 버그 수정
- 강화 비용 배율 작동 X
- 강화 파괴 확률 하락 조정
- 애니메이션 실행 중 버튼이 눌리지 않게 변경
- 강화 완료 하고 5분이 지나면 강화된 무기만 띄우기(버튼 제거)
- 무기 & 맞짱 밸런스 조정 -> 무기 강화 별로 승률 지금보다 더 확 뛰게 설정
