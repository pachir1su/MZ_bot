# MZ_bot

Discord 도박 봇

## 주요 기능
- 10분마다 기본 코인 지급
- 출석 보상, 송금 및 잔액 조회
- 베팅과 순위 확인 등 경제 시스템
- 관리자 전용 설정 및 질의응답 기능
